# Holotools

Holotools is the code base used by the Holobiome team

## Installation

Use the package manager [pip]

```bash
pip install lamp
```

## Usage

```python
from lamp.sentences import word_stats, word_cloud # trimming algorithms

word_cloud(word_stats(['This is one','This is two']))
```
